## ALCOR-decoder 

Files pretaining to decoding and calibrating ALCOR ASIC data, part of dRICH component of ePIC. This version of the files is obsolete. Contains:
-calib.cpp: Program that takes in a binary file of ALCOR data and a parameter (.txt) file; outputs a binary file of ALCOR data with calibrated time
-ccheck.cpp: Checks a calibrated binary file against a raw ALCOR data file
-ctest.cpp: auxillary test file
